# 平平无奇的加载效果

## 代码演示

### 基本用法
```jsx
import {Loading} from 'gzc-ui';

export default () => <Loading lColor='red' rColor='blue' />
```
## API

| 属性     | 说明      | 类型             | 默认值    |
|--------|---------|----------------|--------|
| lColor | 一种球的颜色  | '#fff'/'red'   | '#fff' |
| rColor | 另一种球的颜色 | '#fff'/'red'   | '#000' |
